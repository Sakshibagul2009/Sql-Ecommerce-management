Project Description :-

The E-Commerce Management System is a SQL-based database project designed to manage and organize online shopping operations. It includes customer management, product handling, order processing, payments, shipping, and sales analysis.

 Features :-
Manage product categories and product inventory
Store customer details and registration data
Track customer orders and order items
Handle payment records and payment status
Manage shipping and delivery tracking
Perform CRUD operations on database records
Execute joins, subqueries, and analytical SQL queries
 Database Tables :-
Categories
Products
Customers
Orders
Order_Items
Payments
Shipping

 Technologies Used :-
SQL / MySQL
Database Management System

 How to Run :-
Open MySQL Workbench / SQL Environment
Import the SQL file
Execute the script step by step
View tables and query outputs

 SQL Concepts Covered :-
DDL Commands (CREATE, DROP)
DML Commands (INSERT, UPDATE, DELETE)
CRUD Operations
Joins (INNER, LEFT, RIGHT, FULL)
Subqueries
Aggregate Functions
Window Functions
CASE Statements
Date and String Functions
 Learning Purpose :-

This project helps in understanding:

Relational Database Design
SQL Query Writing
Data Analysis using SQL
Database Relationships with Foreign Keys
Real-world E-commerce Database Structure
